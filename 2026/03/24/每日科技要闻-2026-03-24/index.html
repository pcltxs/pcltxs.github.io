每日科技要闻 - 2026 年 3 月 24 日

📱 人工智能与硬件

iPhone 17 Pro 成功运行 400B 大语言模型

苹果公司最新旗舰 iPhone 17 Pro 被演示成功运行参规模达 4000 亿的大型语言模型，标志着移动端 AI 推理能力的重大突破。该演示展示了手机芯片在本地运行超大模型的能力，无需云端连接即可完成复杂 AI 任务。

Meta 开发 AI CEO 助手

据《华尔街日报》报道，Meta CEO 马克·扎克伯格正在训练一个 AI 代理助手来帮助他完成日常工作。该 AI 助手目前仍处于开发阶段，已能帮助扎克伯格更快地获取信息，例如检索原本需要通过多层人员才能获取的答案。

Mozilla 推出 AI 编程代理问答平台 Cq

Mozilla AI 发布了 Cq 平台，被称为”AI 编程代理的 Stack Overflow”。该平台专为 AI 编程代理设计，帮助开发者解决在使用 AI 辅助编程时遇到的问题。

🛰️ 航天与通信

Amazon Leo 加速卫星发射计划

亚马逊的卫星互联网项目 Amazon Leo（原 Project Kuiper）宣布将年发射任务增加一倍以上，达到 20 多次。目前已部署超过 200 颗卫星，下一次发射定于 3 月 29 日进行。该项目是 SpaceX Starlink 的主要竞争对手。

FCC 更新涵盖设备清单纳入外国制造路由器

美国联邦通信委员会（FCC）更新了涵盖设备清单，将外国制造的消费级路由器纳入监管范围。此举旨在加强网络安全监管。

🔒 网络安全

Trivy 再次遭遇攻击：GitHub Actions 标签被篡改

安全公司 Socket.dev 报告称，流行的安全扫描工具 Trivy 再次成为攻击目标，其 GitHub Actions 标签遭到广泛篡改。这是 Trivy 近期第二次遭遇类似攻击。

黑客伪造 8000 万美元 USR 稳定币

一名黑客成功铸造了价值 8000 万美元的伪造 USR 稳定币，并将其兑换为 ETH。该事件暴露了稳定币协议的安全漏洞。

美国政府要求私营企业”反击黑客”

《经济学人》报道，美国政府正在指示私营企业在遭受网络攻击时采取”反击”措施，这一政策引发了关于网络战规则和企业责任的广泛讨论。

💻 软件与开发

LocalStack 归档 GitHub 仓库并要求账户才能运行

热门本地 AWS 云服务模拟工具 LocalStack 突然归档了其 GitHub 仓库，并要求用户必须创建账户才能继续使用该工具，引发开发者社区强烈反响。

发现正则表达式匹配性能问题的根本原因

研究人员发现，查找所有正则表达式匹配的操作一直以来都存在 O(n²) 的时间复杂度问题，这一发现可能影响大量依赖正则表达式的系统性能。

Bombadil：Web UI 属性测试框架

Antithesis HQ 开源了 Bombadil，一个专为 Web UI 设计的属性测试框架，帮助开发者更有效地测试前端应用的边界情况。

🎮 游戏与娱乐

英国将建 Minecraft 主题公园

Chessington World of Adventures 宣布将建设 Minecraft 主题公园，包括新的过山车、”互动冒险”和积木风格的景点。预计明年开业。同时，布宜诺斯艾利斯将于 5 月开放 Minecraft Experience: Moonlight Trail 体验展。

Xbox 发布重大消息

微软 Xbox 部门发布了多项重要新闻，具体内容待进一步披露。

📷 消费电子

Insta360 推出便携式 SSD

Insta360 与 Lexar 合作推出了售价 199.99 美元的 Quick Reader 512GB 便携式 SSD。该设备可直接连接 Insta360 X5、X4 Air 和 Ace Pro 2 相机进行录制，无需 microSD 读卡器即可传输素材。

TTArtisan 首款即时成像相机

TTArtisan 发布了其首款即时成像相机，采用复古折叠风箱镜头设计，融合了传统摄影美学与现代技术。

早期 Segway 原型机即将拍卖

Segway 早期原型机”Ginger”正在进行拍卖。这款原型机高 52 英寸，包含早期版本的倾斜转向机制，由 Segway 前总裁兼 CEO James Norrod 收藏。拍卖将于本周结束。

🚁 无人机与交通

Google 旧无人机项目在湾区启动服务

Google 的 Wing 无人机配送项目宣布在旧金山湾区启动商业服务，标志着该项目在经过多年测试后正式投入运营。

拉瓜迪亚机场发生飞机与地面车辆相撞事故

两架飞机与地面车辆在纽约拉瓜迪亚机场相撞，导致两名飞行员丧生。事故原因正在调查中。

🌐 操作系统与隐私

GrapheneOS 拒绝强制年龄验证

隐私-focused 的 Android 分支系统 GrapheneOS 明确表示不会强制用户进行年龄验证。开发者在 X 平台上表示：”GrapheneOS 将继续供全球任何人使用，无需提供个人信息、身份证明或账户。如果因法规原因无法在某些地区销售 GrapheneOS 设备，那也只能如此。”

立法者不希望 VPN 阻碍在线年龄验证

美国立法者正在推动法案，要求 VPN 服务不能成为在线年龄验证的障碍。这一提议引发了隐私权倡导者的担忧。

💼 商业与科技

OnlyFans 所有者去世

OnlyFans 母公司 Fenix International Limited 的所有者 Leonid Radvinsky 去世，享年 43 岁。死因是与癌症的长期斗争。Radvinsky 于 2018 年收购了该成人内容平台。

沃尔玛：ChatGPT 结账转化率比网站低 3 倍

Search Engine Land 报道，沃尔玛测试发现使用 ChatGPT 进行结账的转化率比传统网站低 3 倍，表明 AI 聊天界面在电商场景仍有改进空间。

美国与 TotalEnergies 达成近 10 亿美元协议终止海上风电项目

美国政府与法国能源公司 TotalEnergies 达成近 10 亿美元的协议，终止多个海上风电项目。这一决定引发了对可再生能源政策的讨论。

🔧 开源项目

Dune3d：参数化 3D CAD 应用

开源社区发布了 Dune3d，一款参数化 3D CAD 应用程序，为工程师和设计师提供免费的 3D 设计工具。

复古终端音乐播放器 CLIamp

开发者 bjarneo 发布了 CLIamp，一款受 Winamp 启发的复古终端音乐播放器，将经典界面带入命令行环境。

Claude Code 速查表

社区制作了 Claude Code 速查表，帮助开发者更高效地使用这一 AI 编程工具。
