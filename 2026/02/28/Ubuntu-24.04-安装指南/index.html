1. 准备工作

1.1 系统要求

项目 最低要求 推荐配置
CPU 2 GHz 双核 4 GHz 四核或更高
内存 4 GB 8 GB 或更高
硬盘 25 GB 50 GB 或更高
显示器 1024×768 1920×1080 或更高

1.2 备份重要数据

⚠️ 安装前务必备份！

  • 个人文件（文档、图片、视频等）
  • 浏览器书签和密码
  • 应用程序配置
  • 系统许可证和激活码

1.3 准备工具

  • 空白 U 盘（至少 8 GB）
  • 稳定的网络连接
  • 电源适配器（笔记本建议接通电源）

2. 下载安装镜像

2.1 官方下载渠道

访问 Ubuntu 官方网站下载：

2.2 选择版本

  • Ubuntu 24.04 LTS - 长期支持版（推荐）
  • Ubuntu 24.10 - 最新版（短期支持）

LTS 版本提供 5 年安全更新，适合生产环境。

2.3 校验镜像完整性

下载完成后，建议校验 SHA256 值：

与官网公布的哈希值对比，确保一致。

3. 创建启动盘

3.1 使用 Rufus（Windows）

  1. 下载 Rufus：https://rufus.ie
  2. 插入 U 盘
  3. 打开 Rufus，选择：
    • 设备: 你的 U 盘
    • 启动类型: 选择下载的 ISO 文件
    • 分区方案: GPT（UEFI 电脑）或 MBR（旧电脑）
    • 目标系统: UEFI 或 BIOS
  4. 点击”开始”，等待完成

3.2 使用 BalenaEtcher（跨平台）

  1. 下载 BalenaEtcher：https://www.balena.io/etcher
  2. 打开软件，依次操作：
    • Flash from file: 选择 ISO 镜像
    • Select target: 选择 U 盘
    • Flash!: 开始写入
  3. 等待完成，自动校验

3.3 使用 dd 命令（Linux/macOS）

⚠️ 警告: dd 命令会覆盖目标设备全部数据，请确认设备名正确！

4. 安装 Ubuntu

4.1 从 U 盘启动

  1. 插入 U 盘，重启电脑
  2. 进入 BIOS/UEFI 设置（通常按 F2、F12、Del 或 Esc）
  3. 设置 U 盘为第一启动设备
  4. 保存并退出，从 U 盘启动

4.2 启动菜单

选择以下选项之一：

  • Try or Install Ubuntu - 试用或安装（推荐）
  • Ubuntu (safe graphics) - 安全图形模式（显卡兼容性问题时使用）

4.3 选择语言

  1. 左侧列表选择 中文（简体）
  2. 点击 安装 Ubuntu

4.4 键盘布局

  • 选择：Chinese → Chinese
  • 测试键盘输入，确认无误
  • 点击 继续

4.5 更新和其他软件

建议都勾选，减少后续配置步骤。

4.6 安装类型（关键步骤）

选项 A: 清除整个磁盘并安装 Ubuntu（全新安装）

⚠️ 会删除磁盘所有数据！

适合：

  • 新电脑
  • 已备份数据
  • 不再需要原系统

选项 B: 安装 Ubuntu 与现有系统共存（双系统）

适合：

  • 保留 Windows 或其他系统
  • 需要多系统启动

安装程序会自动调整分区大小。

选项 C: 其他选项（手动分区）

适合高级用户，可自定义：

挂载点 建议大小 说明
/ 30-50 GB 系统根目录
/home 剩余空间 用户数据
swap 等于内存大小 交换空间（可选）
/boot/efi 512 MB EFI 引导分区（UEFI 必需）

4.7 选择时区

  • 地图点击 Shanghai 或搜索 Asia/Shanghai
  • 确认时间正确
  • 点击 继续

4.8 创建用户

字段 说明 示例
你的名字 显示名称 Admin
计算机名 网络标识 admin-pc
用户名 登录名 admin
密码 登录密码 ••••••••

⚠️ 密码建议:

  • 至少 8 位
  • 包含大小写字母、数字、符号
  • 妥善保存，重置较麻烦

4.9 开始安装

  1. 确认所有设置无误
  2. 点击 现在安装
  3. 确认分区更改，点击 继续
  4. 等待安装完成（约 15-30 分钟）

5. 首次启动配置

5.1 重启系统

安装完成后，点击 立即重启，按提示拔出 U 盘。

5.2 登录系统

  1. 选择用户
  2. 输入密码
  3. 首次登录可能需要等待系统初始化

5.3 欢迎向导

Ubuntu 24.04 可能显示欢迎向导：

  • 隐私设置: 选择是否启用位置服务、问题报告等
  • 在线账户: 可登录 Google、Microsoft 等账户（可选）
  • Livepatch: 启用内核热补丁（推荐，需要 Ubuntu One 账户）

5.4 网络连接

  1. 点击右上角网络图标
  2. 选择 Wi-Fi 网络并输入密码
  3. 或连接有线网络（通常自动识别）

5.5 显示设置

  • 调整分辨率（推荐原生分辨率）
  • 设置缩放比例（高 DPI 屏幕）
  • 配置多显示器（如有）

5.6 输入法配置

  1. 安装中文语言包（如未安装）
  2. 添加输入法：IBus 或 Fcitx5
  3. 推荐安装 ibus-pinyin 或 fcitx5-pinyin

8. 故障排查

8.1 无法从 U 盘启动

解决方法:

  1. 检查 BIOS/UEFI 启动顺序
  2. 禁用 Secure Boot（安全启动）
  3. 重新制作启动盘（尝试不同工具）
  4. 更换 U 盘或 USB 接口

8.2 安装过程中黑屏/花屏

解决方法:

  1. 启动时按 e 编辑启动参数
  2. 在 quiet splash 后添加 nomodeset
  3. 按 F10 启动
  4. 安装后更新显卡驱动
